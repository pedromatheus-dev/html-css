Principais formatações

Negrito / Destaque

Nessa frase temos uma Termo em negrito usando a tag <b> (sem semântica)

Nessa frase temos uma Termo em destaque usando a tag <strong> (com semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (sem semântica)

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

No outro texto, temos outro texto marcado

Texto grande e pequeno

Estamos criando um texto grande com a tag <big> e um texto pequeno com a tag <small>

Texto deletado

Podemos marcar um texto excluído para indicar que ele deve ser lido, mas não considerado usando a tag <del>

Texto inserido

Podemos marcar um texto inserido para dar ênfase e indicar que ele foi adicionado depois usando a tag <ins>

Existe também o sublinhado com a tag <u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 usamos a tag <sup>

Texto subscrito

Para inserir coisas do tipo H20 usamos a tag <sub>
